# thermine-raspi

## Prerequisites

Read and Write files locker (semaphore):<br />
python3 -m pip install -U readerwriterlock

gRPC:<br />
python -m pip install grpcio<br />
python -m pip install grpcio-tools<br />

SSH:<br />
pip3 install paramiko<br />

S9 only:<br />
pop install toml<br />
